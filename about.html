QADOC 소개

QADOC은 테스트케이스(TC)와 기획서를 한 곳에서, 표준화된 기준으로 작성·검토하도록 돕는 문서 도구입니다. 작성자마다 양식과 기준이 달라 생기는 가독성·파악 시간 편차를 줄이고, 누가 작성하든 공통 기준으로 작성·검토되도록 가이드라인을 시스템화하는 것을 목표로 합니다.

주요 기능

  • 프리셋 기반 작성 — TC·기획서용 템플릿으로 폼에서 작성
  • 룰베이스 검토 — 필수항목 누락·형식·모호 표현을 검사하고 개선 가이드를 제시 (비용 0, 항상 동작)
  • LLM 보조 검토 — 문장 명확성·모호성 등 룰로 못 잡는 부분 보조 (선택)
  • 내보내기 — 테스트케이스는 Excel, 기획서는 PPT로 저장
  • 저장·버전 이력·공유 — 작성 결과 저장, 버전 관리, 읽기전용 공유 링크
  • 검토 기준 편집 — 검토 룰을 화면에서 추가/수정하고 버전으로 관리

현재 상태

현재 QADOC은 내부 파일럿 단계입니다. 데이터는 기본적으로 사용자 브라우저에 저장되며(localStorage), 별도 서버 계정 없이 바로 사용할 수 있습니다.
