QADOC 소개

QADOC은 테스트케이스(TC)와 기획서를 한 곳에서, 표준화된 기준으로 작성·검토하도록 돕는 문서 도구입니다. 작성자마다 양식과 기준이 달라 생기는 가독성·파악 시간 편차를 줄이고, 누가 작성하든 공통 기준으로 작성·검토되도록 가이드라인을 시스템화하는 것을 목표로 합니다.

주요 기능

  • 프리셋 기반 작성 — TC·기획서용 템플릿으로 폼에서 작성, 회사 양식 JSON 업로드 지원
  • 룰베이스 검토 — 필수항목 누락·형식·모호 표현 검사 + 개선 가이드 (비용 0, 항상 동작)
  • LLM 보조 검토 — 문장 명확성·모호성 등 룰로 못 잡는 부분 보조 (선택, 무료 티어)
  • 내보내기 — 테스트케이스는 Excel, 기획서는 PPT
  • 저장·버전 이력·공유 — 저장 시 버전 적립, 되돌리기, 읽기전용 공유 링크
  • 검토 기준 편집 — 룰을 화면에서 추가/수정하고 버전으로 관리·전환

현재 상태

QADOC은 이전 사이드 프로젝트 QASS의 연장선에서 만들어진 내부 파일럿입니다. 데이터는 기본적으로 사용자 브라우저(localStorage)에 저장되며, 별도 서버 계정 없이 바로 사용할 수 있습니다.
