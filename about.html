QADOC 소개

QADOC은 이전 자료의 증적을 바탕으로 기획서와 테스트케이스의 일관성을 점검하고, 어디를 보완하면 좋을지 조언하는 문서 도구입니다. 작성자마다 양식과 기준이 달라 생기는 편차를 줄이고, 검토 기준을 팀이 직접 쌓아 회사의 노하우로 시스템화하는 것을 목표로 합니다.

주요 기능

  • 일관성 점검 · 조언 — 기획서 요구사항이 연결된 TC로 충분히 검증되는지 점검하고 보완 지점을 안내
  • 쌓이는 검토 기준 — 기준을 평범한 문장으로 추가·수정하고 버전으로 관리, 팀의 노하우가 그대로 자산이 됨
  • 기본 틀 · 예시 제공 — TC·기획서 기본 틀과 채워진 예시를 라이브러리에서 바로 열어 시작
  • 자동 검토 + LLM 보조 — 필수항목 누락·형식·모호 표현 검사 + 개선 가이드, 문장 명확성은 LLM이 보조
  • 내보내기 · 공유 · 이력 — TC는 Excel, 기획서는 PPT / 읽기전용 공유 링크 / 저장 시 버전 적립

데이터 보관

작성 내용은 기본적으로 사용자 브라우저(localStorage)에 저장되며, 별도 서버 계정 없이 바로 사용할 수 있습니다. 팀 공용 저장이 필요한 경우 서버 저장소로 전환할 수 있도록 설계되어 있습니다.
